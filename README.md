## Oppgave

Finne anagram i fil med et ord på hver linje.

    $ ./anagram.py -h
    Usage: anagram.py [-h] FILE

    Print all permutations of anagrams in FILE.

    Options:
    --version   show program's version number and exit
    -h, --help  show this help message and exit
